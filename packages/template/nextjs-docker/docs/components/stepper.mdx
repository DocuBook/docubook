---
title: Stepper
description: A component used to display step-by-step instructions directly within the markdown render.
---

In this guide, we utilize custom `Steps` and `Step` components, specifically designed for DocuBook, to display step-by-step instructions directly within markdown.

<Note type="danger" icon="GitBranchMinus" title="Deprecated in v2">
  Legacy tags `<Stepper>` and `<StepperItem>` are deprecated in v2 and kept only for migration compatibility. Use `<Steps>` and `<Step>` for new content.
</Note>

## Preview

<Steps>
  <Step title="Step 1: Clone the DocuBook Repository">
    ```bash
    git clone https://github.com/DocuBook/docubook.git
    ```
  </Step>
  <Step title="Step 2: Access the Project Directory">
    ```bash
    cd docubook
    ```
  </Step>
  <Step title="Step 3: Install Required Dependencies">
    ```bash
    pnpm install
    ```
  </Step>
</Steps>
