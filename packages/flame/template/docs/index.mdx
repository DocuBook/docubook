title: DocuBook Flame 🔥
description: Fast as flame — a Bun-native framework for modern documentation.
---

**@docubook/flame** is a lightweight runtime for documentation sites using React, MDX, and filesystem routing. Runs on Bun, Node.js, and Deno.

```bash
npm create flame@latest my-docs
cd my-docs
npm run dev
```

Check the sidebar for guides on configuration, components, routing, and deployment.
